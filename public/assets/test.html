Welcome!

Does this page look like an OOTB WordPress installation? If you think so, then my goal is partially achieved! In fact, the page is not generated by WordPress, but from a static HTML file available here. If you read the source, you'll find it only contains this "Welcome" paragragh. Then where does the header (all stuff before this paragragh) and footer (all stuff afterwards) come from? The answer is this JS block: 
<script type="text/javascript" src="/index.js"></script>
<script type="text/javascript">
    unippear({
        "headerContainer": "#wrapper",
        "footerContainer": "#wrapper"
    });
</script>
Factoring header, footer and associated assets (CSS, JS, img etc) out of a document isn't new. Almost all template engines are capable of. But those template engines are platform specific and thus only work for one website. Unipear turns the layout into a service to share among multiple websites. If this is of interest to you, return to Unippear to read more.
